Ⅰ. 컴퓨팅 시스템 요약 정리

1. 네트워크의 이해와 종류

  • 네트워크의 개념
    • 서로 다른 기기들이 연결된 통신망.
    • 컴퓨터 네트워크: 유선 또는 무선 통신망으로 연결된 다양한 컴퓨팅 기기.
  • 유선 네트워크
    • 특징: 광케이블, 이더넷 케이블 등을 사용하여 물리적으로 연결.
    • 장점: 안정적이고 빠른 데이터 전송.
    • 단점: 장소 제약 있음.
    • 주요 장비:
      • 허브 및 스위치: 여러 컴퓨팅 시스템 연결, 데이터 전송.
      • 라우터: 네트워크와 네트워크 연결, 데이터 전송 경로 지정 (내부-외부 네트워크).
  • 무선 네트워크
    • 특징: 무선 주파수를 사용하여 연결.
    • 장점: 장소 제약 없이 자유로운 사용, 사물 인터넷(IoT)을 가능하게 함.
    • 주요 기술:
      • 무선 AP (액세스 포인트): 유선 LAN과 무선 LAN 연결, Wi-Fi 인터넷 사용 가능.
      • 방화벽: 외부 불법 접근 및 악의적인 공격 방어 (보안 유지).
      • 전자 태그 (RFID): 소형 반도체 칩 데이터를 무선 주파수로 읽음. (예: 교통카드, 하이패스)
      • 와이파이 (Wi-Fi): 가까운 거리 무선 데이터 통신. 저렴하고 빠름.
      • 블루투스 (Bluetooth): 10m 이내 근거리 무선 통신. (예: 무선 이어폰) → 페어링 과정 필요.
      • 근거리 무선 통신 (NFC): 10cm 이내 비접촉 무선 통신. (예: 스마트폰 결제, 출입 통제)
      • 디스플레이 공유 기술 (미러링): 서로 다른 기기 화면 공유. (예: 미라캐스트, DLNA)

2. 네트워크 환경 구성 및 활용

  • 인터넷 연결
    • 랜 카드 연결 또는 Wi-Fi 사용 후 운영체제에서 네트워크 환경 설정.
    • IP 주소: 인터넷에서 사용하는 주소 (IPv4, IPv6). 윈도우에서 cmd → ipconfig로 확인 가능.
  • 네트워크에서의 공유
    • 다른 컴퓨터 시스템의 데이터 접근 및 프린터, 파일 서버 등 장치 공유 가능.
    • 개인용 컴퓨터 공유: 폴더나 프린터를 공유 설정.
    • 클라우드 컴퓨팅: 시간과 장소에 상관없이 네트워크로 연결된 컴퓨팅 자원을 서비스 형태로 사용. 여러 사람 협업 가능.

3. 사물 인터넷 (IoT)

  • 개념
    • 일상생활 속에서 다양한 물건이 인터넷에 연결되어 데이터를 주고받는 시스템.
    • 컴퓨터 외에 센서와 통신 장치가 있는 각종 사물들이 네트워크로 연결된 환경.
  • 사물의 범위
    • 컴퓨팅 기기(컴퓨터, 태블릿)뿐 아니라 스피커, 냉장고, 전등, TV 등 주변의 모든 구체적인 존재.
  • 피지컬 컴퓨팅
    • 컴퓨터와 현실 세계를 연결하는 시스템.
    • 구성:
      • 센서 (입력 장치): 온도, 밝기, 동작 등 주변 상황 인식.
      • 마이크로컨트롤러 (처리 장치): 센서 정보 처리 및 액추에이터 제어.
      • 액추에이터 (출력 장치/구동기): 마이크로컨트롤러의 제어에 따라 물리적으로 동작.
  • 활용 사례
    • 스마트워치 건강 관리, 실시간 버스 위치 추적.
    • 의료: 환자 건강 상태 실시간 모니터링.
    • 건설: 드론, 로봇, 센서를 활용한 효율 및 안전 관리.
    • 스마트홈: 조명, 온도, 가전제품 등 원격 제어 및 상호 연결.
    • 교통: 자율주행 자동차, 지능형 교통 시스템.

시험 출제 예상 포인트:

  • 네트워크 종류: 유선/무선 네트워크의 특징, 장단점, 주요 장비(허브, 스위치, 라우터)의 역할 비교.
  • 무선 네트워크 기술: RFID, Wi-Fi, 블루투스, NFC의 특징과 실생활 적용 사례를 구분하여 설명. 특히 페어링의 의미.
  • IP 주소: 개념 및 확인 방법.
  • 클라우드 컴퓨팅: 개념, 장점, 활용 사례.
  • 사물 인터넷 (IoT): 개념, '사물'의 의미, 피지컬 컴퓨팅의 구성 요소(센서, 마이크로컨트롤러, 액추에이터)와 각 역할, 그리고 다양한 실생활 활용 사례.
  • 각 기술의 장점과 단점, 활용 예시를 명확히 이해하는 것이 중요합니다.
