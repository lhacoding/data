Ⅲ. 프로그래밍 요약 정리

3. 프로그래밍의 이해

4. 제어구조의 응용

1. 변수와 자료형

  • 변수(Variable)
    • 정의: 프로그래밍에서 값을 저장하는 기억 공간.
    • 값 저장: 등호(=)를 이용하여 변수에 값을 저장하고, 저장된 값을 변경할 수 있음.
    • 형식: 변수명 = 변수에 저장할 값 (상수, 변수, 수식 등)
    • 변수명 지정 규칙:
      • 알파벳 대소문자, 한글, 언더바(_), 숫자만 사용 가능.
      • if, for 등과 같은 파이썬 예약어는 변수명으로 사용할 수 없음.
      • 알파벳 대소문자는 구별해서 사용 (a와 A는 다름).
      • 첫 글자에 숫자가 올 수 없음.
  • 자료형(Data Type)
    • 정의: 프로그램에서 사용하는 데이터의 종류. 데이터의 종류에 따라 메모리에 저장되는 방식이나 연산 방법이 달라짐.
    • 파이썬의 주요 자료형:
      • 정수형(Integer): 소수점이 없는 숫자. (예: 10, -5)
      • 실수형(Float): 소수점이 있는 숫자. (예: 3.14, -0.5)
      • 문자열(String): 문자들의 나열. 따옴표(' ' 또는 " ")로 묶음. (예: "Hello", '파이썬')
      • 불리언형(Boolean): 참(True) 또는 거짓(False)만을 나타내는 자료형. 조건문의 결과 등에 사용.
    • 자료형 확인: type() 함수 사용. (예: type(10) 결과는 )
    • 자료형 변환: 특정 자료형으로 변경.
      • int(): 정수형으로 변환.
      • float(): 실수형으로 변환.
      • str(): 문자열로 변환.

2. 연산과 연산자

  • 연산자(Operator)
    • 정의: 데이터를 처리하여 결과를 얻는 데 사용되는 기호.
    • 산술 연산자: 사칙연산. (+, -, *, /, %(나머지), //(몫), **(거듭제곱))
    • 할당 연산자: 변수에 값을 저장 (=) 또는 연산 후 할당 (+=, -=, *= 등).
    • 비교 연산자: 두 값을 비교하여 참(True) 또는 거짓(False) 반환. (==, !=, <, >, <=, >=)
    • 논리 연산자: 여러 조건을 조합하여 참/거짓 판단. (and, or, not)
  • 연산자 우선순위: 연산자마다 수행되는 순서가 다름 (산술 > 비교 > 논리). 괄호 ()를 사용하면 우선순위를 높일 수 있음.

3. 입출력

  • 출력(Output)
    • print() 함수: 데이터를 화면에 출력.
    • 형식 지정:
      • 콤마(,)로 여러 값 출력 시 띄어쓰기.
      • sep: 출력 값들 사이 구분자 지정 (예: print(1, 2, sep='-') → 1-2).
      • end: 출력 끝에 추가될 문자 지정 (기본값은 개행 문자). (예: print('a', end=' ')).
      • f-스트링 (f-string): 문자열 안에 변수 값을 바로 포함하여 출력. (예: name = "김", print(f"이름: {name}"))
  • 입력(Input)
    • input() 함수: 사용자로부터 데이터를 입력받음.
    • 특징: input() 함수로 입력받은 값은 항상 문자열 자료형으로 처리됨.
    • 자료형 변환 필수: 숫자를 입력받아 계산할 경우 반드시 int()나 float() 등으로 형 변환 필요. 
      num1 = int(input("첫 번째 숫자: "))

4. 조건문

  • 조건문(Conditional Statement)
    • 정의: 특정 조건에 따라 코드 블록의 실행 여부를 결정하는 문법.
    • if 문: 
      if 조건식:
    # 조건식이 참일 때 실행되는 코드
    • if-else 문: 
      if 조건식:
    # 조건식이 참일 때 실행되는 코드
else:
    # 조건식이 거짓일 때 실행되는 코드
    • if-elif-else 문: 
      if 조건식1:
    # 조건식1이 참일 때 실행
elif 조건식2:
    # 조건식1이 거짓이고 조건식2가 참일 때 실행
else:
    # 모든 조건식이 거짓일 때 실행
    • 들여쓰기(Indentation): 파이썬에서 코드 블록을 구분하는 중요한 요소. 반드시 동일한 깊이로 들여쓰기 해야 함. (일반적으로 공백 4칸 사용)

5. 반복문

  • 반복문(Loop Statement)
    • 정의: 특정 코드 블록을 여러 번 반복하여 실행할 때 사용.
  • for 문:
    • 정의: 정해진 횟수만큼 반복하거나 특정 순서(리스트, 튜플, 문자열 등)의 요소를 하나씩 꺼내면서 반복.
    • range() 함수: 숫자 범위를 생성.
      • range(n): 0부터 n-1까지.
      • range(start, end): start부터 end-1까지.
      • range(start, end, step): start부터 end-1까지 step 간격으로.
    • 형식: 
      for 변수 in 시퀀스(리스트, 튜플, 문자열, range 등):
    # 반복할 코드
  • while 문:
    • 정의: 특정 조건이 참인 동안 반복해서 실행. 조건이 거짓이 되면 반복 종료.
    • 형식: 
      while 조건식:
    # 조건식이 참인 동안 반복할 코드
    • 무한 루프 방지: while문 내부에 조건식을 거짓으로 만들 수 있는 코드가 반드시 있어야 함. (예: 반복 횟수를 세는 변수 증가)
  • 반복 제어문:
    • break: 반복문을 즉시 종료하고 반복문 다음 코드로 이동.
    • continue: 반복문의 현재 반복을 건너뛰고 다음 반복을 시작.

6. 자료구조

  • 자료구조(Data Structure)
    • 정의: 데이터를 효율적으로 저장하고 관리하는 방법.
  • 리스트(List)
    • 특징: 여러 개의 값을 순서대로 저장하는 자료형. []로 묶음.
    • 가변(Mutable): 한 번 생성된 후에도 값을 추가, 삭제, 변경 가능.
    • 다양한 자료형 저장: 하나의 리스트에 여러 자료형의 값을 함께 저장할 수 있음.
    • 인덱싱(Indexing): 순서(인덱스)를 통해 특정 값에 접근 (0부터 시작). (예: my_list[0])
    • 슬라이싱(Slicing): 리스트의 일부분을 잘라내어 새로운 리스트 생성. (예: my_list[1:3])
    • 주요 메서드:
      • append(): 리스트 맨 뒤에 요소 추가.
      • insert(인덱스, 값): 특정 인덱스에 요소 삽입.
      • remove(값): 특정 값 삭제 (첫 번째 발견된 값).
      • pop(인덱스): 특정 인덱스의 요소를 삭제하면서 반환 (인덱스 없으면 마지막).
      • del 리스트[인덱스]: 특정 인덱스의 요소 삭제.
      • len(): 리스트의 길이(요소 개수) 반환.
  • 튜플(Tuple)
    • 특징: 여러 개의 값을 순서대로 저장하는 자료형. ()로 묶음.
    • 불변(Immutable): 한 번 생성되면 값 추가, 삭제, 변경 불가.
    • 활용: 변경되면 안 되는 데이터를 저장할 때 사용.
  • 딕셔너리(Dictionary)
    • 특징: 키(Key)와 값(Value)의 쌍으로 데이터를 저장하는 자료형. {}로 묶음.
    • 순서 없음: 값에 접근할 때 인덱스 대신 키를 사용.
    • 가변(Mutable): 생성 후 키-값 쌍 추가, 삭제, 변경 가능.
    • 활용: 이름-나이, 단어-의미 등 대응 관계 데이터 저장.
    • 주요 메서드:
      • 딕셔너리[키] = 값: 키에 해당하는 값 변경 또는 새로운 키-값 쌍 추가.
      • 딕셔너리.keys(): 모든 키 반환.
      • 딕셔너리.values(): 모든 값 반환.
      • 딕셔너리.items(): 모든 키-값 쌍 반환.
      • del 딕셔너리[키]: 특정 키-값 쌍 삭제.
  • 데이터 구조의 차원 (dimension)
    • 데이터 구조를 구성하는 요소의 수.
    • 1차원: 리스트처럼 하나의 선형 구조.
    • 2차원: 행과 열로 구성 (예: 표, 흑백 이미지).
    • 3차원: 행, 열, 높이로 구성 (예: 컬러 이미지 - R,G,B 색상 정보).

시험 출제 예상 포인트:

  • 변수: 정의, 값 저장 방법, 변수명 지정 규칙 (예약어, 첫 글자 숫자 등)
  • 자료형: 정수, 실수, 문자열, 불리언의 특징과 예시, type() 함수, 자료형 변환 함수(int(), float(), str()).
  • 연산자: 산술, 할당, 비교, 논리 연산자의 종류와 역할, 연산자 우선순위.
  • 입출력: print() 함수의 사용법(sep, end, f-string), input() 함수의 특징(항상 문자열 반환)과 형 변환의 필요성.
  • 조건문: if, if-else, if-elif-else 문의 구조와 실행 흐름, 들여쓰기의 중요성.
  • 반복문: for 문과 while 문의 사용법 및 차이점, range() 함수, break와 continue의 역할.
  • 자료구조:
    • 리스트: 정의, 특징(가변, 순서), 인덱싱/슬라이싱, 주요 메서드(append, insert, remove, pop, len).
    • 튜플: 정의, 특징(불변, 순서), 리스트와의 차이점.
    • 딕셔너리: 정의(키-값 쌍), 특징(순서 없음, 가변), 값 접근 방법, 주요 메서드(keys, values, items).
    • 데이터 구조의 차원: 1차원, 2차원, 3차원 데이터의 개념과 예시 (특히 이미지 표현).
  • 각 파이썬 코드 예시를 읽고 결과를 예측하거나, 특정 기능을 수행하는 코드를 작성할 수 있도록 연습.
