Ⅱ. 데이터 요약 정리

1. 데이터의 이해

  • 데이터의 중요성
    • 다양한 유형의 정보(기상 정보 등)는 예보 외에 고속도로 사고 위험 예측, 태양광 발전량 예측, 관광 코스 추천, 수산물 어획량 예측 등 다양한 분야에 활용되어 삶의 편리함과 합리적 의사 결정에 기여.
    • 단순히 정보를 모으는 것을 넘어, 다양한 분석을 통해 새로운 가치 창출.
  • 빅데이터
    • 기존 데이터베이스 관리 도구로는 수집, 저장, 관리, 분석하기 어려운 대량의 정형 또는 비정형 데이터.
    • 특징 (3V):
      • 규모(Volume): 데이터 양이 매우 많음.
      • 속도(Velocity): 데이터 생성, 처리, 변화 속도가 매우 빠름.
      • 다양성(Variety): 정형 데이터와 함께 비정형 데이터(텍스트, 이미지, 영상, 음성 등) 포함.

2. 데이터의 종류와 특징

  • 정형 데이터
    • 미리 정해진 형식과 구조에 따라 저장되고 관리되는 데이터.
    • 특징: 고정된 필드에 저장, 일관된 형식, 주로 데이터베이스나 스프레드시트 형태.
    • 예시: 이름, 전화번호, 회원 가입 정보, 상품 판매량, 온도, 습도 등.
  • 비정형 데이터
    • 미리 정해진 형식이나 구조가 없는 데이터.
    • 특징: 유동적인 형식, 분류 및 분석이 어려움.
    • 예시: 이미지, 오디오, 비디오, SNS 게시글(텍스트), 웹 문서, 이메일 등.

3. 데이터의 처리 및 관리

  • 데이터 처리 과정:

    데이터 수집 → 데이터 저장 → 데이터 가공/분석 → 데이터 표현

    • 데이터 수집: 센서, 카메라, 웹 스크래핑 등.
    • 데이터 저장: 적절한 형식과 장소에 보관.
    • 데이터 가공/분석: 의미 있는 정보로 변환, 패턴 추출 (정렬, 필터링, 통계 분석, 기계 학습 등).
    • 데이터 표현: 그래프, 차트, 시각화 자료 등으로 나타냄.
  • 데이터의 압축: 데이터의 중복성을 제거하여 파일 크기 줄임.
    • 목적: 저장 공간 절약, 데이터 전송 시간 단축.
    • 압축 방식:
      • 무손실 압축: 압축 후 복원 시 원본과 100% 동일. 데이터 손실 없어 정확성 중요 시 사용. (예시: ZIP, PNG, GIF, BMP)
      • 손실 압축: 압축 시 원본 일부 제거, 복원 시 원본과 완전히 같지 않음. 용량 효율성 중요 시 사용. (예시: JPG, MP3, MP4) → 압축률 높을수록 품질 저하 심함.
  • 데이터의 암호화: 데이터를 알아볼 수 없는 형태로 변환하여 정보 유출이나 오용 방지.
    • 목적: 데이터의 기밀성, 무결성, 가용성 보장.
    • 활용: 개인 정보 보호, 금융 거래 보안, 통신 보안.

4. 데이터의 활용

  • 데이터 시각화
    • 복잡한 데이터를 그림, 그래프, 차트 등으로 표현하여 쉽게 이해하고 분석.
    • 목적: 데이터의 패턴, 추세, 이상치 등을 빠르게 파악.
    • 예시: 막대그래프, 꺾은선그래프, 원그래프, 산점도, 워드 클라우드.
  • 빅데이터 활용 사례
    • 기상 예측: 정확한 날씨 예보 및 어획량 예측, 사고 위험 예측.
    • 의료: 환자 데이터 분석을 통한 질병 예측, 맞춤형 치료법 개발.
    • 교통: 실시간 교통량 분석을 통한 최적 경로 안내, 교통 체증 완화.
    • 마케팅: 고객 구매 이력 분석을 통한 맞춤형 상품 추천, 타겟 마케팅.
    • 도시 문제 해결: CCTV, 센서 데이터 분석으로 범죄 예방, 재난 관리, 스마트 도시 구축.

시험 출제 예상 포인트:

  • 빅데이터의 3V 특성 (규모, 속도, 다양성) 정확히 이해.
  • 정형 데이터와 비정형 데이터의 특징과 예시 구분.
  • 데이터 처리 과정(수집-저장-가공/분석-표현)의 각 단계 역할.
  • 데이터 압축의 목적, 무손실/손실 압축의 차이점, 각 방식의 특징과 대표적인 파일 형식(JPG, MP3, ZIP, PNG 등) 숙지.
  • 데이터 암호화의 목적과 중요성.
  • 데이터 시각화의 중요성 및 다양한 표현 방식.
  • 실생활에서의 빅데이터 활용 사례들을 구체적으로 연결하여 설명.
